PS3Muxer - mkv video to PlayStation 3 / PlayStation 4 / XBox One converter

PS3Muxer is a new very simple program which will remux mkv file with h.264 video and AC3/DTS/LPCM/AAC audio to a M2TS file which is playable on the Sony PlayStation3.

No video transcoding, optional audio transcoding, delay and fps detection, external audio tracks, batch processing, multi-audio,
possibility of splitting a file into parts, possibility of changing a h.264 profile and AC3 bitrate, Windows, Linux, Mac OS X, Free source code.
