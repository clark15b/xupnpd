About xupnpd API

The xupnpd API accepts requests via standard HTTP GET, and can respond with plain text.

Add feed

wget -q -O - 'http://127.0.0.1:4044/ui/api?action=add_feed&plugin=youtube&feed=thelonelyisland&name=The+Lonely+Island'
OK

Show feeds

wget -q -O - 'http://127.0.0.1:4044/ui/api?action=feeds'
1;Vimeo HD Channel
2;HD Xtreme sports
3;Mountain Bike Channel
4;YouTube Top Rated
5;GT - PS3 - Review
6;GT - PS3 - Preview
7;GT - PS3 - Gameplay
8;GiantBomb - All
9;AG - New
10;IVI - New
11;The Lonely Island

Remove feed by id

wget -q -O - 'http://127.0.0.1:4044/ui/api?action=remove_feed&id=11'
OK

Save feeds to persist

wget -q -O - 'http://127.0.0.1:4044/ui/api?action=save_feeds'
OK

Reload feeds

wget -q -O - 'http://127.0.0.1:4044/ui/api?action=update_feeds'
OK

Reload playlists

wget -q -O - 'http://127.0.0.1:4044/ui/api?action=reload'
OK

Show active streams

wget -q -O - 'http://127.0.0.1:4044/ui/api?action=status'
22578;127.0.0.1 Battlefield 1943 Theme Song

Stop stream by pid

wget -q -O - 'http://127.0.0.1:4044/ui/api?action=kill&pid=22578'
OK
